Cómo publicar un módulo en npm

Primero ejecutamos en la consola el comando 'npm init -y' para crear un archivo package.json que nos sirve para describir un proyecto o un módulo que podemos publicar en npm

Luego creamos una carpeta src y dentro creamos un archivo index.js. Este archivo nos va a servir para poder exportar algunas funciones y luego cuando instale este módulo en otros proyectos poder importar estas funciones.

Entonces creamos dos funciones llamadas greet y users.
